## zh-CN

支持上传一个文件夹里的所有文件。

## en-US

You can select and upload a whole directory.

````jsx
import { Upload, Button, Icon } from 'antd';

ReactDOM.render(
  <Upload action="//jsonplaceholder.typicode.com/posts/" directory>
    <Button>
      <Icon type="upload" /> Upload Directory
    </Button>
  </Upload>,
  mountNode);
````
